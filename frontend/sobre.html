Sobre o Projeto FéTech

Conheça a motivação, objetivos e equipe por trás deste Trabalho de Conclusão de Curso


Objetivo do Projeto

O FéTech é um sistema de organização pastoral desenvolvido como Trabalho de Conclusão de Curso (TCC) para o curso de Informática para Internet do Instituto Federal Catarinense.

Motivação

A proposta do projeto nasceu da observação das dificuldades encontradas na Paróquia São José Operário para organizar informações, comunicar decisões e coordenar atividades pastorais. Atualmente, muitos processos ainda dependem de registros manuais, grupos de mensagens e documentos dispersos, o que pode dificultar o planejamento e o engajamento dos membros.

Buscamos oferecer uma solução tecnológica que:

  • Centralize informações da comunidade
  • Facilite a comunicação entre membros
  • Organize escalas de voluntários
  • Disponibilize recursos espirituais
  • Promova o engajamento dos fiéis

Contexto Acadêmico

Este projeto foi desenvolvido como requisito para conclusão do curso de [Nome do Curso] no Instituto Federal Catarinense, campus [Nome do Campus].

O trabalho envolve pesquisa, desenvolvimento e implementação de um sistema web completo, demonstrando as competências técnicas adquiridas ao longo do curso, incluindo:

Frontend

Desenvolvimento de interface responsiva com HTML5, CSS3 e JavaScript, garantindo experiência do usuário otimizada.

Backend

Arquitetura de servidor, banco de dados e APIs para gerenciamento eficiente das informações.

UX/UI

Design centrado no usuário, considerando acessibilidade e usabilidade para diferentes perfis de usuários.
